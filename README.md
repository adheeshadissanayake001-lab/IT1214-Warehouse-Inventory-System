# Warehouse Inventory Management System

Simple console-based Warehouse Inventory System using Java Collections Framework (ArrayList).

## Features
- Add new item
- Remove item
- Update quantity
- Search by ID or Name
- Display all items

## How to Run
1. Compile all files:
   ```bash
   javac Warehouse.java
2. Run the program:
   java Warehouse
